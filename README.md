## Списки

Чтобы добавить ненумерованные списки, необходимо выделить пункты звёздочками (*).
Например, вот так:

* Элемент 1
* Элемент 2

Чтобы добавить нумерованние списки, необходимо просто пронумеровать, либо ставить (1). Например, вот так:

1. Первый пункт
1. Второй пункт
